# Security Compliance Toolkit

This repository provides automated dependency scanning and security compliance checks for GitHub repositories.

## Installation

```bash
git clone https://github.com/herdiyana256/security-compliance-tool
cd security-compliance-tool
```

## Usage

Open this folder with your AI assistant (GitHub Copilot, Claude Code) and ask it to scan the repository.

The `.mcp.json` file configures the GitHub integration in read-only mode.
